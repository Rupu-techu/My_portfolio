# AI & ML Portfolio

A production-ready portfolio built with **Next.js 15**, **TypeScript**, **Tailwind CSS**, and **Framer Motion**.

## Quick Start

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Personalise

All content lives in **`src/lib/data.ts`** — edit the exported objects.

| What               | Where                          |
|--------------------|--------------------------------|
| Name, bio, links   | `personal` object              |
| Story & interests  | `about` object                 |
| Projects           | `projects` array               |
| Work history       | `experiences` array            |
| Awards & certs     | `achievements` array           |
| Profile photo      | Replace `◈` in `Hero.tsx` with `<Image>` and drop `photo.jpg` in `/public` |
| Resume             | Drop `resume.pdf` in `/public` |
